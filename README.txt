1. portarsi nella cartella docker all'interno del progetto
2. lanciare il comando "docker compose build"
3. lanciare il comando "docker compose up -d"
4. se necessario fornire i permessi 777 da bash allo script init.sh, "chmod -R 777 init.sh"
5. da bash lanciare lo script ./init.sh
6. per avviare i test PHPUnit se necessario fornire i permessi 777 da bash allo script test.sh, "chmod -R 777 test.sh"
7. da bash lanciare lo script ./test.sh
